# Calculator MX Support Website

This repository contains a simple static support website for the iOS app Calculator MX.

## Pages

- `index.html` - Support page
- `privacy.html` - Privacy Policy page

## Enable GitHub Pages

In GitHub, open this repository and go to:

Settings -> Pages -> Build and deployment -> Deploy from a branch -> main branch -> root folder.

After saving, GitHub will publish the site and provide a public URL that can be used in App Store Connect.

## Notes

This site uses plain HTML and CSS only. It does not include analytics, tracking, cookies, forms, databases, external scripts, dependencies, or a build step.
